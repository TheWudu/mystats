# mystats
Small rails application to render stats nicely

# Usage

Use [mongo_cpp](https://github.com/TheWudu/mongo_cpp/) to import sessions to the database.

Start rails 

```
rails s
```

and visit

```
http://localhost:3000/charts/index
```
